Latitude Project Summary

The purpose of this project was to analyze weather changes as you approach the equator. Using OpenWeatherMap API to randomly select over 500 cities as our dataset.

With the dataset, we used Matplotlib to plot different aspects of the weather at different latitudes. Visualizations include: Max temperature, humidity, cloudiness, and wind speed. This site contains the source data, plots, and analysis.
